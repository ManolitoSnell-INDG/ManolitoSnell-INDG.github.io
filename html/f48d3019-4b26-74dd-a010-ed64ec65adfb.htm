ParseNetkeyBasketError
Parses a netkey basket error object. Note that one error object can contain errors for one or more products with the same error type (such as OUT_OF_STOCK, etc.). Example 1) For one or more insufficient stock products the error object looks like: { "message": " .... ","errorKey":"MS_LOW_STOCK_ERROR", "errorParams":"22108280001:3 , 22108740001:2" } The "errorParams" contains the string-list of productId:quantityAvailable-pairs; "22108280001:3,22108740001:2". Example 2) For two or more out of stock product errors: { "message": " .... ","errorKey":"MS_OUT_OF_STOCK_ERROR", "errorParams":"22108280001 , 22108740001" } The "errorParams" now contains a string-list of productIds; "22108280001 , 22108740001".
