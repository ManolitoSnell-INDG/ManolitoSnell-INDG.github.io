BoxMap(Mesh, Single, Single, Single)
Calculates uv coordinates for the specified mesh, similar to 3DS Max boxmapping. A planar mapping is applied for each of the six faces fo the cube, where the vertex normal determines which face to use. Has to look up the underlying vertices and normals from the mesh. Use BoxMap(Vector3[] theVertices, Vector3[] theNormals, float aWidth, float aHeight, float aDepth) instead when possible. This method is not thread-safe!
BoxMap(Vector3, Vector3, Single, Single, Single)

CalculateMeshTangents(Mesh)
Calculate the tangets for the given mesh. This uses the properties UnityEngine.Mesh.{triangles,vertices,uv,normals} creating a deep copy of each. When possible use already cached references to these arrays instead, calling the overloaded version of this method. This method is not thread-safe!
CalculateMeshTangents(Vector3, Int32, Vector2, Vector3)
Calculates the mesh tangents. This overload is created to avoid needles lookups of Mesh properties. Each property lookup allocates a new deep copy of the underlying array, causing gigantic GC alloc overhead. This method is not thread-safe!
