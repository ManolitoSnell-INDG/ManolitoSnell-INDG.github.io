FindMatchingGroupEndIndex
Find the matching end index for a delimited set of characters which could be nested. For example the string "((foo) + bar)", has 2 groups delimited by the "(" and ")" characters. Searching the matching end index for the first group would result in the position at the end of the string (the last matching ")" character) While searching for the matching end index of the embedded group would result in the position right behind "foo".
