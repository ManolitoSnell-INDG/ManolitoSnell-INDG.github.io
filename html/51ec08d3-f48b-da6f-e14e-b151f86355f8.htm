The RoomsPageMediator type exposes the following members.

Fields
  NameDescription
COMPONENT_EVENT_OPEN_AR
Request to initiate the AR flow.
COMPONENT_METHOD_ADD_ROOM
Adds a room to the component. aData:String, JSON, the new item data: { "id": "1", "name": "Room 1", "imageUrl": "hex://image1.jpg" "thumbUrl": "hex://thumb1.jpg" }
COMPONENT_METHOD_CLEAR_ROOMS
Removes all rooms from the component.
COMPONENT_METHOD_HIDE_COMPONENT
Hides the component. aTransitionType:int, the type of transition to use, see the constants defined below.
COMPONENT_METHOD_PREPARE_TO_SHOW
Prepare the component to be shown. Should be called in the pre-state-change phase before the component is shown. aShowImmediate:Boolean, indicates if the component should be shown immediately, for transition from initial or entry page.
COMPONENT_METHOD_REMOVE_ROOM
Removes the specified room from the component. aId:String, the identifier of the room that was removed.
COMPONENT_METHOD_SET_ROOMS
Sets the data for all rooms. aData:String, JSON, the room data: { "currentRoomId":2, "caretEnabled":true, "rooms" : [ { "id": "1", "name": "Room 1", "imageUrl": "hex://image1.jpg" "thumbUrl": "hex://thumb1.jpg" }, { "id": "2", "name": "Room 2", "imageUrl": "hex://image2.jpg" "thumbUrl": "hex://thumb2.jpg" } ] }
COMPONENT_METHOD_SHOW_COMPONENT
Shows the component. aTransitionType:int, the type of transition to use, see the constants defined below.
COMPONENT_METHOD_UPDATE_ROOM
Updates the data of the specified room in the component. aId:String, the id of the room to update. aData:String, JSON, the new item data: { "id": "1", "name": "Room 1", "imageUrl": "hex://image1.jpg" "thumbUrl": "hex://thumb1.jpg" }
COMPONENT_NAME
The name of the UI component.
DEFAULT_AR_IMAGE
The image which is being used for the AR room (when applicable).
DEFAULT_AR_THUMBNAIL
The image which is being used for the AR thumbnail (when applicable).
DEFAULT_AR_UID
The identifier used to denote that the room is used for AR mode.
EXTERNAL_EVENT_CUSTOMER_NAME_CHANGED
The name of the customer has changed. { id:String, name:String }
EXTERNAL_EVENT_DELETE_ROOM
Delete the specified room. aRoomId:String, the id of the room to delete.
EXTERNAL_EVENT_DUPLICATE_ROOM
Duplicate the specified room. aRoomId:String, the id of the room to duplicate.
EXTERNAL_EVENT_GO_TO_ENTRY_PAGE
Go to the entry page. No parameters.
EXTERNAL_EVENT_KEYBOARD_HIDE
Hide the keyboard. aAnimate:Boolean
EXTERNAL_EVENT_KEYBOARD_SHOW
Display the keyboard. { text:String, caret:int, submitLabel:String }
EXTERNAL_EVENT_NEW_ROOM
Create a new room. No parameters.
EXTERNAL_EVENT_OPEN_ROOM
Open the specified room. aRoomId:String, the id of the room to open.
EXTERNAL_EVENT_READY_TO_SHOW
Prepare-to-show is complete, the component is ready to be shown (or already visible if specified).
EXTERNAL_EVENT_ROOM_NAME_CHANGED
The label of the room has changed. { id:String, title:String }
EXTERNAL_EVENT_SEARCH_AND_SORT_ROOMS
Sort rooms. { sortField:String, sortDirection:String, searchString:String}
m_mediatorName
The mediator name
(Inherited from Mediator.)
m_viewComponent
The view component being mediated
(Inherited from Mediator.)
NAME
The default mediator name.
ROOM_LOAD_TIMEOUT
The maximum time in seconds to wait for the scene to complete loading before the panel is animated out.
TRANSITION_HIDE_FADE
Hide the component with a simple fade transition. For the transition to shape selection. Should be done in the state-change phase.
TRANSITION_HIDE_IMMEDIATE
Hide the component without a transition. For the transition to entry page. Should be done in the state-change-ended phase.
TRANSITION_HIDE_TO_3D
Hide the component with a transition to the 3d scene. For the transition to any of the 3d states. Should be done in the state-change phase.
TRANSITION_HIDE_TO_AR
Hide the component with a transition to the AR scene. For the transition to any of the 3d states. Should be done in the state-change phase.
TRANSITION_SHOW_FADE
Show the component with a simple fade transition. For the transition from shape selection. Should be done in the state-change phase.
TRANSITION_SHOW_FROM_3D
Show the component with a transition from 3d scene to the corresponding room. For the transition from any of the 3d states. Should be done in the state-change phase.
TRANSITION_SHOW_FROM_AR
Show the component with a transition from AR scene to the corresponding room. For the transition from any of the 3d states. Should be done in the state-change phase.
TRANSITION_SHOW_IMMEDIATE
Show the component without a transition. For the transition from init and entry page. Should be done in the pre-state-change phase.
