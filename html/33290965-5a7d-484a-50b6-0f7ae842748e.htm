SnappingManager
The SnappingManager is the access point to perform snapping on elements during a transform. At the start of a transform, the method 'initializeTransform' should be called with the elements being transformed as argument. During the transform the method 'performSnapping' can be called to perform snapping for one or more elements from the initial set.
SnappingResult
The SnappingResult class describes the result of a snapping operation executed by a SnapToHelper object.
SnappingUtils
The SnappingUtils class holds a number of static utility methods that are used for snapping.
SnapToHelper
The SnapToHelper class is an abstract base class for objects that are used by the SnappingManager to perform the actual snapping.
