The ContextRegisteredObject type exposes the following members.

Fields
  NameDescription
InjectGlobal
If true, the object's members are reachable as global identifiers.
IsLocalObject
Indicates that this is the 'local object' whose instance is not registered beforehand. If true, the Object field is ignored.
Name
The global-level identifier that can be used to access this object. If this value is not a valid identifier (e.g. by the first character being a space, or being the empty string) the object can only be reached via global injection.
Object
The instance of the class, or a Type for static access.
Type
The type of the instance. Also determines if static or instance access is used.
