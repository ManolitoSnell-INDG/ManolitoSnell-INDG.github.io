Properties

AffectsParentModel
If true (eg for windows), the model of the tie-parent should be updated if the tie-child is modified (if it is tied to a tie-plane). Default is false
ClassIds (Inherited from ElementIdData.)
ElementIds (Inherited from ElementIdData.)
HasGravity
True if the associated tie-child has gravity, ie when it falls down (eg a door). False if the tie-child only sticks to its tie-parent (eg a window or picture). If false, the user can determine the z-position of the tie-child. Default is true.
HasIds (Inherited from ElementIdData.)
IsMandatory
True if the associated Element must tie itself to a tie-parent, false if this is optional.
KeepInTiePlane
If true (eg for windows and pictures), the associated tie-child cannot be placed outside of the tie-plane to which it is tied (if it is tied to a tie-plane). Default is false.
LimitTiePointToPlaneExpression
If false, the tie point of the tie-child may end up anywhere, even outside a tie plane. Default is true.
SnapDistance
The maximum distance to a tie-parent when the associated tie-child automatically snaps to a tie-parent while being dragged.
TiePoints
The tie-points definition.
