Returns a cubic function to evaluate movement equation p(t) = p0 + v0 * t + a0 * t^2 / 2 + j0 * t^3 / 6, witch v(t) = v0 + a0 * t + j0 * t^2 / 2, a(t) = a0 + j0 * t, j(t) = j0. where p0, p(endTime), v0, and endTime are given, such that v(endTime) = 0.
