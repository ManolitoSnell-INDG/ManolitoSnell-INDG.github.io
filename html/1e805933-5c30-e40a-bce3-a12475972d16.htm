ExpressionType Class
This class is used within the parser/expression system to communicate the compile-time type of an expression. Outside the parser system you should never need to use this class.

Callable
Indicates that the current type is actually the result of an expression that returned a method. The only valid action on a callable type is to execute a call on it.
Dynamic
Single instance of an ExpressionType representing a run-time typed expression. Prefer to use this instance when representing a dynamic type.
Instance
If the Instance is not null this is assumed to be the actual instance of the object to work on, which is used for PropertyHolder resolution - at compile time we cannot always reliably determine which members are in a property holder, and at runtime we need to figure out if an identifier actually exists on an instance.
PropertyTyper
A PropertyTyper instance that can be used to determine types for IPropertyHolder implementors. If the current Type does not implement IPropertyHolder this value will never be used. For classes that do, this value may still be null.
