A MaterialRequest describes the properties of a requested material, it is used to find and instantiate a matching material implementation. Instances can be used as Id in IManagedResource / key in ResourceManager.
MaterialSemantic
Class containing constants for the default material property semantics. The semantics are used to provide a common interface to the framework shaders. A MaterialImplementor is used to link the semantics with the respective shader properties. Integers are used to allow for project specific semantics.
