ChainUtilsGetMinimumSnapDistance Method
Returns the minimum snap distance to snap the specified list of ChainElements, after moving them according to 'aRelativeMoveVector', or float.MaxValue if there is no snap vector within the specified 'aSnapAngle'. If not null, 'aOutSnapChainPoint' will be set to the chain point that snapped or invalid if there is no snap vector.
