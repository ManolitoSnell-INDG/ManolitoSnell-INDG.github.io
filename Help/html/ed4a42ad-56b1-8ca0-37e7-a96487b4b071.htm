MeasurementConverter Class
Converts a set of Measurement instances as generated by the MeasurementGenerator to Measurement instances as consumed by the MeasurementRenderer. These are currently different types due to the existence of HxVector3D. Once the HxVector is factored out, both measurement classes can probably be unified.
