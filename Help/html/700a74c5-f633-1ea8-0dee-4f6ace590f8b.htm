XmlModelUtilsParseRotation Method
Parses a single rotation vector. Rotations are specified in degrees and in clockwise direction around the respective axis (x = right axis, y = up axis, z = forward axis). The rotation is performed in xzy order, to be compatible with the hx backend (not the default zxy order).
