BaseStateChangeMediator
A base mediator for mediators of components that need to respond to state changes. Usually a derived mediator will need its ShowApplicationStates list defined on creation (see below).
MainStateChangeMediator
A basic state change mediator that shows and hides its component in the main phase of application state change.
PrePostStateChangeMediator
An basic implementation that hides the component before the main transition takes place. And shows the component after the main transition has been performed.
