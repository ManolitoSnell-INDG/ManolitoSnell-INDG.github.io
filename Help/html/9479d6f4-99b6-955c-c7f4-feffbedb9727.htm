AlwaysIncludeBoundsClassIds

ChainPointOffsetExpression
The vertical offset where the chain point is located, relative to the center. The default value is "0".
ChainSets
The IDs of the ChainSets for which to include the ChainPoints when searching for closed chains in the set.
HasFixedOrientation
True if the orientation of the corresponding chain line should not be changed when a connected chain element is transformed. Default is false.
HasUpperSide
True if the chain element has an upper and lower part, ie when 'lowerPartOffetExpression' != null. Default is false. The parts have the same - horizontal - orientation has the chain points.
IgnoreBoundsClassIds
The IDs of the ElementDefinitionClasses that should ignore the bounds of the associated ChainElement when determining their bounds.
LowerPartOffsetExpression
The vertical offset where the lower part of the chain element starts, relative to the center. If null, the chain element has no separate upper and lower parts. Default is null.
