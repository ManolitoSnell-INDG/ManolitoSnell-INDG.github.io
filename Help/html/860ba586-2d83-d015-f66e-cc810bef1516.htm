LocalizationProxy Class
Owner of all localization data. Dispatches several notifications. LOCALE_CHANGED: When the current locale has been changed. LOCALE_LOAD_SUCCESS: When the load of a localization file succeeded, this notification is always encompassed with at LOCALE_CHANGED notificiation. LOCALE_LOAD_FAILURE: When the load of a localization file failed for some reason.
