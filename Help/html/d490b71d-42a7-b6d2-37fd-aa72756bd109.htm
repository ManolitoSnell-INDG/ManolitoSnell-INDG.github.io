Members
  Member nameValueDescription
Play0 Start the play back of the effect. When it is already playing the command is ignored.
Pause1 Pause the effect when it is already playing.
TogglePause2 Toggles the playback mode, meaning when the effect not being played the effect is being played. And when the effect is playing, the effect is being paused.
Stop3 Stop the current effect, leaving the values as they are at that moment.
ToggleStop4 Toggles the playback mode, meaning when the effect not being played the effect is being played. And when the effect is playing, the effect is being stopped.
End5 End the current effect, no matter what state it is in. The effect is sort of stopped and fast forwarded to the end of the effect.
ToggleEnd6 Toggles the playback mode, meaning when the effect is not being played the effect is being played. And when the effect is playing, the effect is being stopped.
