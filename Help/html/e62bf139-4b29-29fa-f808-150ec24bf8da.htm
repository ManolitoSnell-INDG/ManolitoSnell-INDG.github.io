The radius of the circular area, in mm. This will be converted to pixels based on the device dpi.

Type: SystemInt32
The number of subdivisions along the radius, this controls the number of samples taken.

Return Value

Type: ListElement
The selectable elements. The list is sorted; objects in the front have highest priority.
Remarks
- Objects closer to the center of the touch get higher priority. - In build mode the structural elements have higher priority than regular elements; in furnish mode it's the other way round. - To prevent selecting through an object, only objects that are actually closer to the camera than the touch-center are considered for a priority increase. - By default objects are sorted according to their hit-distance, closest objects are listed first.
