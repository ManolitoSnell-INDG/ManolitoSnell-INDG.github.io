Explicit
Tell whether the creation of the room is explicit or not. When it is explicit the end user initiated an explicit request to create a new room. When it is not explicit, the end user triggered an action which lead to the creation of a new room For example, while clicking on the application logo, the intendend behaviour (at this moment) is to go home and start a new room. Aka the end user did not ask to create a new room, but the flow dictates that a new room should be created.
