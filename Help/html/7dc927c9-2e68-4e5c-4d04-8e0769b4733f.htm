IndexedExpression Class
Evaluates indexing an instance, either because the instance is an array, or because it has an indexer method defined. Note that current indexing support is limited to one argument, so multidimensional arrays or multi-argument indexers are not supported. This implementation checks the DefaultMemberAttribute of a class for the name of the indexer, so indexers with non-default names work.
