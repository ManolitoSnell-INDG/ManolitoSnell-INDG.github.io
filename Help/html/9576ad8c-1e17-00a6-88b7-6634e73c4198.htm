SetCeilingHeightCommand Class
Will try to set the current ceiling height which may involve in the extrusion of walls (when they are defined like that). It is not guarentied that the requested height is actually set, the actual height depends on whether it is allowed by all the elements on the stage and there relationship with each other. To get the actual changed height (if, at all) start listening for
Copy
HexNotes.CEILINGHEIGHT_CHANGED
notifications.
