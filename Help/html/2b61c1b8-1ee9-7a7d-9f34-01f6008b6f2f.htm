SnapUtilsGetSnapPoint Method
Determines if 'aSnappingPoint' can snap to one of the lines between it and the points in 'aConnectedPoints'. If so, the snapped point is returned. Otherwise null is returned. Snapping is done to the X-axis and/or the Y-axis.
